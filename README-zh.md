# pet-admin

> 这是一个vue admin 管理后台 它只包含了 Element UI & axios & iconfont & permission control & lint，这些搭建后台必要的东西。


## Build Setup

``` bash

# Clone project


# Install dependencies
npm install


# serve with hot reload at localhost:9528
npm run dev

# 打包正式环境
npm run build:prod

# 打包测试环境
npm run build:sit
```
